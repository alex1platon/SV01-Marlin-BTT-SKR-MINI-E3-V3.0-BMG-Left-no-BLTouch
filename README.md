# SV01-Marlin-BTT-SKR-MINI-E3-V3.0-BMG-Left-no-BLTouch
Marlin firmware for the Sovol SV-01 (original) 3D printer upgraded with a BMG/Left extruder, no autolevel probe, and the BTT SKR Mini E3 V3.0 control board

The firmware is tested on my machine.  Config files are provided for reference.  Source code from Marlin GitHub https://github.com/MarlinFirmware/Marlin
